# ❤️ Heart Disease Prediction

## 📌 Overview
This project predicts whether a person is likely to have heart disease based on medical parameters using a Machine Learning model. The model analyzes patient information and provides a prediction that can assist in early health assessment.

---

## 🚀 Features
- Predicts the likelihood of heart disease.
- User-friendly interface for entering patient details.
- Trained using a supervised Machine Learning algorithm.
- Displays prediction results instantly.
- Uses real-world heart disease dataset for training.

---

## 📂 Dataset
The project uses the **Heart Disease Dataset**, which contains medical information such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise Induced Angina
- Oldpeak
- Slope
- Number of Major Vessels
- Thalassemia
- Target (Heart Disease)

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

---

## 🤖 Machine Learning Model
The project uses the **Logistic Regression** algorithm for binary classification.

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/heart-disease-prediction.git
```

Go to the project folder:

```bash
cd heart-disease-prediction
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
